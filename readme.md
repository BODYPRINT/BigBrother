BigBrother
===========


Compilation
-----------

For compilation, you need :

* JDK 6 (Sun JDK or OpenJDK Highlight recommanded)
* Install [Maven 3](http://maven.apache.org/download.html)
* Check out this repo and run `mvn clean install`
